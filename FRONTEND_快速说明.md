# 前端快速说明

## 页面定位

这是一个面向小学数学 AI 教学平台的首页工作台，核心是“输入问题 -> 调用后端 -> 展示结果”。

## 版式结构

1. 左侧窄栏：仅做视觉分区，当前无功能按钮。
2. 中间主区域：欢迎语、输入区、案例区、结果区。
3. 整体风格：浅灰背景、细边框、圆角输入容器，接近你给的草图。

## 输入区功能

1. 问题输入框：支持直接输入数学问题。
2. 模式下拉：`极速 / 平衡 / 深入`（当前为前端展示参数）。
3. 附件按钮：可选择本地文件，显示“已选择附件：文件名”。
4. 发送按钮：点击后请求后端接口。

## 请求与接口

1. 前端请求：`POST /api/lesson-assets`
2. 默认后端地址：`http://127.0.0.1:8000`
3. 请求体：

```json
{
  "grade": 3,
  "question": "你的问题"
}
```

## 结果展示

1. 成功：页面下方显示最近一次回答文本（`answer`）。
2. 失败：显示错误提示（例如 HTTP 状态码或网络失败）。
3. 加载中：禁用输入与按钮，发送按钮显示加载状态。

## 精选案例

1. 页面底部有 6 个案例卡片。
2. 点击任意卡片会自动填充问题并立即发送请求。

## 已实现文件

1. `frontend/src/App.jsx`：页面结构和交互逻辑。
2. `frontend/src/App.css`：样式实现（含响应式）。
3. `frontend/src/index.css`：全局基础样式。
